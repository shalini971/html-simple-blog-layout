HTML Simple Blog Layout

 📌 Description
This project is a simple blog post layout created using HTML. It demonstrates the use of semantic tags and proper structuring of a webpage.

 ✨ Features
- Header with title, date, and author
- Introduction, main content, and conclusion sections
- Image integration
- Clean and structured layout using HTML
